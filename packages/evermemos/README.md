# @prizm/evermemos

本包为 **EverMemOS** 的 **TypeScript 移植与修改版本**，供 Prizm 服务端记忆与检索使用。

- **原项目**：EverMemOS（长期记忆系统）
- **原项目许可证**：Apache License 2.0  
- **原项目版权**：Copyright 2025 EverMem AI

本包在移植过程中对实现与存储方式做了适配与修改（如 LanceDB + SQLite 等），具体以本仓库源码为准。

## 使用

本包为 Prizm monorepo 内部依赖，由 `@prizm/server` 引用。构建与测试见根目录与 [prizm/README.md](../../prizm/README.md)。
